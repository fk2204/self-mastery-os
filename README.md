# Self-Mastery OS

A comprehensive personal development command-line system that helps you systematically improve across all critical areas of life.

## Quick Start

```bash
# Navigate to the directory
cd C:\Users\fkozi\self-mastery-os

# Run the app
python src/main.py

# Or use the batch file
mastery.bat
```

## Daily Commands

```bash
mastery.bat am      # Morning check-in
mastery.bat pm      # Evening reflection
mastery.bat week    # Weekly review
mastery.bat status  # Progress dashboard
```

## Features

- **Onboarding**: Personalized setup based on your goals and constraints
- **Morning Check-in**: Sleep/energy logging, priority setting, action planning
- **Evening Reflection**: Task review, wins/lessons capture, day scoring
- **Weekly Review**: Pattern analysis, module scoring, next week planning
- **Adaptive Coaching**: AI-driven insights based on your patterns
- **Habit Tracking**: Streaks, completions, trend analysis
- **Knowledge Base**: Curated best practices for all 9 life modules

## Data Storage

All data is stored locally in the `data/` folder:
- `user_profile.json` - Your settings and levels
- `habits.json` - Habit definitions and streaks
- `logs/` - Daily check-in logs
- `reviews/` - Weekly reviews
